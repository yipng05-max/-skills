# 中文文献核验清单

## 第一层：存在性验证（必做）

- [ ] **知网搜索**：使用 WebSearch 搜索 `"<论文标题>" site:cnki.net` 确认论文存在
- [ ] **万方搜索**：使用 WebSearch 搜索 `"<论文标题>" site:wanfangdata.com.cn` 交叉验证
- [ ] **百度学术搜索**：使用 WebSearch 搜索 `"<论文标题>" site:xueshu.baidu.com` 补充验证
- [ ] **DOI 验证**（如有）：运行 `verify_doi.py` 确认 DOI 可解析（注意：多数中文文献无 DOI，无 DOI 不代表不存在）
- [ ] **作者-标题匹配**：确认声称的作者与实际论文作者一致

## 第二层：元数据准确性（应做）

- [ ] **发表年份**：确认发表年份与实际记录一致
- [ ] **期刊名称**：确认期刊名称完全正确（注意区分"学报"vs"杂志"vs"期刊"）
- [ ] **卷/期/页码**：交叉核对卷号、期号、页码
- [ ] **作者单位**：核实作者所属机构
- [ ] **ISSN/CN号**：如有提供，验证是否与期刊匹配

## 第三层：核心期刊验证（涉及期刊级别声称时）

- [ ] **北大核心**：在北京大学《中文核心期刊要目总览》中查证
- [ ] **CSSCI（南大核心）**：在中文社会科学引文索引来源期刊目录中查证
- [ ] **CSCD**：在中国科学引文数据库来源期刊目录中查证
- [ ] **科技核心**：在中国科技论文统计源期刊目录中查证
- [ ] **注意年份**：核心期刊目录每隔几年更新，某期刊可能某些年份是核心，其他年份不是

## 第四层：中文图书验证

- [ ] **ISBN 验证**：使用 WebSearch 搜索 `"ISBN <号码>"` 并在国家图书馆 (opac.nlc.cn) 或 WorldCat 查证
- [ ] **出版社验证**：确认出版社存在且出版过该书
- [ ] **版次验证**：确认声称的版次确实存在
- [ ] **豆瓣读书**：使用 WebSearch 搜索 `"<书名>" site:book.douban.com` 查证

## 第五层：学位论文验证

- [ ] **知网学位论文库**：使用 WebSearch 搜索 `"<论文标题>" site:cdmd.cnki.com.cn` 或在知网学位论文分类下搜索
- [ ] **万方学位论文库**：使用 WebSearch 搜索 `"<论文标题>" 学位论文 site:wanfangdata.com.cn`
- [ ] **授予单位核实**：确认该高校确实有相关学科的学位授予权

## 第六层：政策文件与新闻验证

- [ ] **政府文件**：在政府官网搜索发文字号（如"国发〔2023〕XX号"），使用 WebSearch 搜索 `"<发文字号>" site:gov.cn`
- [ ] **新闻报道**：在声称的媒体官网搜索，使用 WebSearch 搜索 `"<标题关键词>" site:<媒体域名>`
- [ ] **人民日报数据库**：如声称为人民日报文章，可搜索 `"<标题>" site:people.com.cn`

## 中文文献验证的特殊注意事项

1. **无 DOI 不等于不存在**：大量中文文献没有 DOI，CrossRef 搜索不到不能直接判定为虚构
2. **中英文标题**：部分中文论文有英文标题，可尝试用英文标题在 CrossRef 搜索
3. **作者姓名格式**：中文作者在英文文献中可能有多种拼写方式（如"张三"→"Zhang San"/"San Zhang"/"S. Zhang"）
4. **知网收录有时滞**：最近发表的论文可能尚未被知网收录
5. **付费墙**：知网、万方等数据库需要付费或机构账号才能获取全文，但搜索和摘要通常免费可见
