# Common Citation Styles Reference

Quick reference for citation and bibliography formatting across major academic styles.

---

## APA 7th Edition

**In-text citation:**
- One author: (Smith, 2020) or Smith (2020)
- Two authors: (Smith & Jones, 2020)
- Three or more: (Smith et al., 2020)
- Direct quote: (Smith, 2020, p. 15)

**Bibliography entry examples:**

Journal article:
```
Author, A. A., & Author, B. B. (2020). Title of article in sentence case. Title of Periodical in Title Case and Italics, 50(3), 123–135. https://doi.org/xxxxx
```

Book:
```
Author, A. A. (2020). Title of work: Capital of subtitle (2nd ed.). Publisher. https://doi.org/xxxxx
```

Book chapter:
```
Author, A. A. (2020). Title of chapter. In E. E. Editor (Ed.), Title of book (pp. 100–150). Publisher.
```

Conference paper:
```
Author, A. A. (2020, March 15–18). Title of paper. Conference Name, Location. https://doi.org/xxxxx
```

## GB/T 7714-2015 (Numeric / 顺序编码制)

China's national standard for bibliographic references. Widely used in Chinese journals.

**In-text citation:**
- Single: [1]
- Multiple: [1,3] or [1-3]
- Superscript style: ^[1]

**Bibliography entry examples:**

Journal article (期刊论文 [J]):
```
[1] 作者1, 作者2. 题名[J]. 刊名, 年, 卷(期): 起止页码.
[1] 张三, 李四. 关于某问题的研究[J]. 某某学报, 2020, 35(2): 100-110.
[2] SMITH A, JONES B. Title of article[J]. Journal Name, 2020, 50(3): 123-135.
```

Book (专著 [M]):
```
[3] 作者. 书名[M]. 版次. 出版地: 出版社, 年: 页码.
[3] 王五. 学术写作指南[M]. 2版. 北京: 高等教育出版社, 2020: 50-60.
```

Conference paper (会议论文 [C]):
```
[4] 作者. 题名[C]//会议论文集名. 出版地: 出版社, 年: 页码.
```

Dissertation (学位论文 [D]):
```
[5] 作者. 题名[D]. 城市: 学校, 年.
```

Standard (标准 [S]):
```
[6] 标准编号, 标准名称[S]. 出版地: 出版社, 年.
```

Online resource (电子文献 [EB/OL]):
```
[7] 作者. 题名[EB/OL]. (发布日期)[引用日期]. URL.
```

**Document type identifiers (文献类型标识):**
| Code | Type |
|------|------|
| M | Monograph / 专著 |
| J | Journal article / 期刊 |
| C | Conference paper / 会议论文 |
| D | Dissertation / 学位论文 |
| R | Report / 报告 |
| S | Standard / 标准 |
| P | Patent / 专利 |
| N | Newspaper / 报纸 |
| EB/OL | Online resource / 电子文献 |

## GB/T 7714-2015 (Author-Year / 著者-出版年制)

**In-text citation:**
- (张三, 2020)
- (Smith, 2020)
- (Smith and Jones, 2020)
- (Smith et al., 2020)
- 张三(2020)认为...

**Bibliography entry examples:**
```
张三, 李四. 2020. 关于某问题的研究[J]. 某某学报, 35(2): 100-110.
SMITH A, JONES B. 2020. Title of article[J]. Journal Name, 50(3): 123-135.
```

Note: Year is placed after authors, before title.

## Vancouver (ICMJE) Style

Used widely in biomedical journals (e.g., NEJM, The Lancet, BMJ).

**In-text citation:**
- Superscript numbers: text^1 or text^1,2 or text^1-3
- Or bracketed: text (1) or text (1,2)

**Bibliography entry examples:**

Journal article:
```
1. Smith AB, Jones CD. Title of article. Journal Abbreviation. 2020;50(3):123-35. doi:xxxxx
```

Book:
```
2. Smith AB. Title of Book. 2nd ed. City: Publisher; 2020.
```

Note: Author names as "Last AB" (no periods in initials), journal names abbreviated, no italics.

---

## IEEE Style

Used in engineering and computer science journals/conferences.

**In-text citation:**
- Bracketed numbers: [1], [1], [2], [1, 3], [1]–[3]

**Bibliography entry examples:**

Journal article:
```
[1] A. B. Smith and C. D. Jones, "Title of article," Journal Name Abbrev., vol. 50, no. 3, pp. 123–135, Mar. 2020. doi: xxxxx.
```

Conference paper:
```
[2] A. B. Smith, "Title of paper," in Proc. Conf. Name, City, Country, 2020, pp. 100–105.
```

Book:
```
[3] A. B. Smith, Title of Book, 2nd ed. City, State: Publisher, 2020.
```

Note: Author names as "A. B. Last", article titles in quotes, journal/book titles in italics, abbreviations for months.

## Chicago Style (Notes-Bibliography / Author-Date)

**Notes-Bibliography (used in humanities):**

Footnote:
```
1. Author First Last, Title of Book (City: Publisher, 2020), 45.
```

Bibliography:
```
Last, First. Title of Book. City: Publisher, 2020.
```

**Author-Date (used in sciences/social sciences):**

In-text: (Smith 2020, 45)

Bibliography:
```
Smith, John. 2020. Title of Book. City: Publisher.
```

---

## MLA 9th Edition

**In-text citation:**
- (Smith 15) — author + page, no comma
- (Smith and Jones 15)
- (Smith et al. 15)

**Works Cited entry examples:**

Journal article:
```
Smith, Adam, and Beth Jones. "Title of Article." Journal Title, vol. 50, no. 3, 2020, pp. 123–35. DOI.
```

Book:
```
Smith, Adam. Title of Book. Publisher, 2020.
```
