## 漏斗型结构

最经典的顶刊文献综述结构，从宏观到微观逐步聚焦：

```
第一层：宏观领域定位（1段）
  → 确立研究所在的大领域，点明领域重要性
  → 引用领域奠基性文献（seminal works）和近期综述

第二层：核心构念/变量的文献脉络（2-4段）
  → 按主题而非时间组织
  → 每个主题段落呈现：共识 → 分歧 → 未解决问题
  → 构念之间用逻辑衔接句串联

第三层：变量间关系的研究现状（1-3段）
  → 聚焦到本文核心关系
  → 呈现已有实证发现的一致性与矛盾性
  → 识别理论解释的不充分之处

第四层：研究空白与本文定位（1段）
  → 明确指出literature gap
  → 说明本文如何填补空白
  → 自然过渡到假设/理论模型
```

## 理论驱动结构

适用于理论贡献导向的研究：

```
第一层：核心理论的介绍与发展脉络（1-2段）
  → 理论起源、核心主张、关键假设
  → 理论在本领域的应用历史

第二层：理论的拓展与批评（1-2段）
  → 已有的理论延伸
  → 理论局限性或边界条件

第三层：理论与本文变量的结合（2-3段）
  → 用理论视角重新审视核心变量
  → 理论如何预测变量间关系

第四层：理论空白与本文理论贡献定位（1段）
  → 理论的未探索区域
  → 本文的理论拓展方向
```

## 争论型结构

适用于存在明显理论分歧的领域：

```
第一层：现象/问题的重要性（1段）
  → 为什么这个问题值得关注

第二层：观点A及支持文献（1-2段）
  → 理论逻辑 + 实证支持

第三层：观点B及支持文献（1-2段）
  → 对立理论逻辑 + 实证支持

第四层：整合与超越（1-2段）
  → 为什么两种观点都不完整
  → 本文提供的整合视角或新解释
```

## 段落内部组织模式

### 综合式段落（Synthesis Paragraph）— 顶刊首选

```
主题句（概括该段核心论点）
  → 综合多项研究支撑论点（Smith, 2020; Lee, 2021; Zhang, 2022）
  → 指出细微差异或条件限制
  → 批判性评价（方法局限/样本局限/理论不足）
  → 过渡句（引出下一段）
```

示例：
"Prior research has consistently demonstrated that transformational leadership
enhances employee creativity (Wang et al., 2019; Liu & Chen, 2020; Park, 2021).
However, these studies predominantly rely on cross-sectional designs and
self-reported measures, leaving the causal mechanism largely unexplored.
Moreover, while intrinsic motivation has been proposed as a key mediator
(Amabile, 1996), recent evidence suggests that this relationship may be
contingent on contextual factors such as organizational climate
(Zhou & George, 2019). This points to the need for..."

### 避免的模式：逐篇罗列式（串珠式）

❌ 错误示范：
"Smith (2020) found that X leads to Y. Jones (2021) examined the relationship
between X and Z. Lee (2022) studied the effect of X on W."

## 段落间衔接模式

### 逻辑递进型衔接
- "Building on this theoretical foundation, subsequent research has extended..."
- "While these findings establish the direct effect of X, less is known about the mechanisms through which..."
- "Despite this growing body of evidence, a critical limitation remains..."

### 转折对比型衔接
- "However, not all scholars agree on this interpretation..."
- "In contrast to the predominantly positive findings, a parallel stream of research suggests..."
- "This optimistic view, however, has been challenged by..."

### 聚焦深入型衔接
- "Of particular relevance to the current study is..."
- "Narrowing the focus to [specific context], the picture becomes more nuanced..."
- "Within this broader framework, [specific variable] has received increasing attention..."
