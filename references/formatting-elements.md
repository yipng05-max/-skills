# Academic Paper Formatting Elements Checklist

Comprehensive reference of all formatting elements to extract from sample papers and apply to manuscripts.

## 1. Page Layout

- **Paper size**: A4 (210mm x 297mm), Letter (8.5" x 11")
- **Margins**: top, bottom, left, right (common: 2.54cm / 1 inch all sides; Chinese journals often 2.5-3cm)
- **Orientation**: portrait / landscape
- **Columns**: single-column, two-column (with column gap)
- **Headers**: running title, author name, journal name, page number position
- **Footers**: page number, journal info, DOI
- **Page numbering**: Arabic, Roman (for front matter), position (center/right)

## 2. Title Block (首页信息区)

- **Title**: font family, size, weight (bold), alignment (center/left), case (title case / sentence case)
- **Subtitle**: font, size, separator style
- **Author names**: font, size, separator (comma / and), superscript markers for affiliations
- **Affiliations**: font, size, format (numbered / lettered / symbol)
- **Corresponding author**: notation style, email format
- **Date info**: received / revised / accepted dates format
- **Funding/acknowledgment placement**: footnote vs end section

## 3. Abstract & Keywords

- **Abstract label**: font, size, bold, position (inline / separate line)
- **Abstract body**: font, size, indentation, line spacing
- **Structured abstract**: sections (Background, Methods, Results, Conclusions)
- **Keywords label**: font, bold
- **Keywords**: font, size, separator (comma / semicolon), capitalization
- **Chinese abstract (中文摘要)**: 摘要标签格式、正文字体字号

## 4. Headings (标题层级)

- **Level 1 (章标题)**: font, size, weight, alignment, numbering (1. / I. / 一、), spacing before/after
- **Level 2 (节标题)**: font, size, weight, alignment, numbering (1.1 / A. / (一)), spacing
- **Level 3 (小节标题)**: font, size, weight, numbering (1.1.1 / 1) / 1.), spacing
- **Level 4**: font, size, weight, numbering scheme
- **Numbering scheme**: Arabic (1, 1.1, 1.1.1) / Chinese (一、(一)、1.) / Roman / None
- **Run-in headings**: heading on same line as body text

## 5. Body Text (正文)

- **Font family**: English (Times New Roman, Palatino, etc.), Chinese (宋体 SimSun, 仿宋 FangSong)
- **Font size**: 10pt, 10.5pt (五号), 12pt (小四号), etc.
- **Line spacing**: single, 1.15, 1.5, double, exact (e.g., 20pt)
- **First-line indent**: 2em (中文) / 0.5in (English) / none
- **Paragraph spacing**: before/after (0pt, 6pt, 12pt)
- **Alignment**: justified / left-aligned
- **Widow/orphan control**: on/off

## 6. Figures (图)

- **Caption position**: below figure
- **Caption label**: "Figure 1" / "Fig. 1" / "图1" — font, size, bold
- **Caption text**: font, size, alignment (center / left)
- **Numbering**: sequential through paper / per-section (Fig. 2.1)
- **Figure placement**: inline / float, width constraints
- **Sub-figure labels**: (a), (b), (c)
- **Source line**: font, size, position

## 7. Tables (表)

- **Caption position**: above table
- **Caption label**: "Table 1" / "表1" — font, size, bold
- **Caption text**: font, size, alignment
- **Three-line table (三线表)**: top border thick, header separator, bottom border thick, no vertical lines
- **Table body font**: size, alignment (center columns / left text)
- **Notes below table**: font size, "Note:" prefix style
- **Column header**: bold, alignment

## 8. Equations (公式)

- **Numbering**: right-aligned, (1) / (2.1)
- **Alignment**: centered
- **Font**: italic for variables, Roman for functions
- **Spacing**: before/after equation

## 9. References / Bibliography (参考文献)

- **Citation style in text**:
  - Numeric: [1], [1,2], [1-3]
  - Author-year: (Smith, 2020), (Smith & Jones, 2020), (Smith et al., 2020)
  - Superscript: ^[1], ^[1,2]
- **Bibliography format**:
  - Numbered list [1] vs hanging indent
  - Author format: Last, F.M. / Last FM / 姓名
  - Title: sentence case / title case / italic
  - Journal name: abbreviated / full, italic
  - Volume: bold / regular
  - Pages: 1-10 / 1–10
  - DOI/URL inclusion
  - Chinese references (GB/T 7714): [序号] 作者.题名[文献类型标识].刊名,年,卷(期):页码.
- **Heading**: "References" / "参考文献" / "Bibliography"
- **Font size**: often smaller than body (e.g., 9pt, 五号)
- **Line spacing**: often tighter than body

## 10. Footnotes / Endnotes (脚注/尾注)

- **Marker style**: superscript Arabic / symbol (*, †, ‡) / circled numbers
- **Font size**: typically 8-9pt (小五号)
- **Separator line**: short rule above footnotes
- **Numbering**: per-page / continuous

## 11. Lists

- **Bullet style**: bullet (•), dash (–), none
- **Numbering style**: (1), 1., a), i.
- **Indentation**: amount, hanging indent
- **Spacing**: between items

## 12. Chinese-Specific Elements (中文特殊排版要素)

- **中文字体**: 宋体(正文)、黑体(标题)、楷体(摘要)、仿宋
- **字号对照**: 初号42pt, 小初36pt, 一号26pt, 小一24pt, 二号22pt, 小二18pt, 三号16pt, 小三15pt, 四号14pt, 小四12pt, 五号10.5pt, 小五9pt, 六号7.5pt
- **首行缩进**: 2字符（2em）
- **标点**: 全角标点、标点挤压
- **参考文献格式**: GB/T 7714-2015
- **中英文混排**: 中文宋体 + 英文 Times New Roman
- **关键词分隔**: 分号（；）
