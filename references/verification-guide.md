# 文献核查指南

## 核查流程

在引用任何文献之前，必须完成以下核查步骤以确保文献的真实性。

### 第1步：文献检索与初步筛选

使用 WebSearch 工具搜索相关学术文献。搜索策略：

1. **关键词组合搜索**：
   - `"[核心概念1]" AND "[核心概念2]" site:scholar.google.com`
   - `"[核心概念]" review OR meta-analysis`
   - `"[核心概念]" [顶刊名称缩写]` (如 AMJ, ASQ, SMJ, JAP, AMR, OS, JOM等)

2. **特定作者搜索**：
   - `[作者姓名] [研究领域关键词] Google Scholar`

3. **引用链追踪**：
   - 找到一篇高度相关的论文后，搜索其参考文献中的关键引用

### 第2步：逐条核实

对每一条拟引用的文献，必须核实以下信息：

| 核查项 | 方法 |
|--------|------|
| 作者姓名 | WebSearch 确认拼写和全名 |
| 发表年份 | WebSearch 确认具体年份 |
| 文章标题 | WebSearch 确认完整标题 |
| 期刊名称 | WebSearch 确认发表在哪个期刊 |
| 核心发现 | WebSearch/WebFetch 确认论文实际研究内容 |

### 第3步：核实搜索模板

对每条文献使用以下搜索模板核实：

```
搜索词: "[作者姓氏] [年份] [文章标题中2-3个关键词] [期刊名]"
```

若搜索无法确认某文献的存在，必须：
1. 尝试变体搜索（如不同拼写、缩写期刊名）
2. 若仍无法确认，**立即移除该文献**，不得在综述中引用
3. 寻找替代文献来支撑相同论点

### 核查状态标记

在写作过程中，对每条文献标记状态：
- ✅ 已核实：搜索确认文献存在且信息准确
- ⚠️ 待确认：部分信息已确认，需进一步核实
- ❌ 不可用：无法确认文献存在，必须移除

### 核查原则

1. **宁缺毋滥**：如果无法100%确认一条文献的真实性，不引用
2. **交叉验证**：重要文献至少通过2个来源确认
3. **内容一致**：确认文献实际研究内容与综述中的引用描述一致
4. **避免幻觉**：绝不编造或猜测作者、年份、期刊等信息
